## ğŸ“¦ Installation

### Requirements

- Python 3.9+
- PostgreSQL (optional, SQLite default)
- Redis (for background jobs)
- Docker (optional)

### Quick Install

```bash
# Clone repository
git clone https://github.com/ITherso/monolith.git
cd monolith

# Create virtual environment
python3 -m venv .venv
source .venv/bin/activate

# Install dependencies
pip install -r requirements.txt
pip install -r requirements_extra.txt

# Initialize database
flask db upgrade

# Start server
make run-prod
```

### Docker Install

```bash
# Build and run
docker-compose up -d

# Access
http://localhost:8080
```
